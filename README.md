# Student Marks Data Analysis

## Dataset
- Source: Kaggle
- Instances: 100
- Attributes:
  - number_courses (Number of Courses Opted)
  - time_study (Average Time Studied per Day)
  - Marks (Marks Obtained)

## Objective
To analyze how study time and course load impact student marks.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Distribution of Marks
- Study Time vs Marks
- Courses vs Marks
- Correlation Heatmap

## Key Insights
- Study time shows a positive correlation with marks.
- Course count has a weaker impact on performance.
